# Panthers Club GAS

Panthers Club のGoogle Apps Script管理用リポジトリです。

## 目的

- PDFスタッツ取込
- Gemini API連携
- PlayerStats_Import から PlayerStats への正式反映
- GoogleスプレッドシートをDBとして活用
- AppSheetによる入力・閲覧UIとの連携

## 使用技術

- Google Apps Script
- Googleスプレッドシート
- AppSheet
- Gemini API
- ChatGPT
- Codex

## 運用方針

- Code.gs はApps Scriptの全文バックアップとして管理する
- 変更時は全文コードを保存する
- エラー発生時は、修正前後のコードをGitHubに残す
